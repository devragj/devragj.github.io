TypeA/TableauA.js

This file stores the classes needed to perform the Robinson-Schensted algorithm (and its inverse) and to draw the results.
The main class is the TableauA class.
The Tile class stores the information about the contents of one square in a tableau.
The TileGrid class is a helper class for the TableauA class.
The TableauAPair class holds a pair of TableauA.
The ParameterRS class handles the input to the Robinson-Schensted algorithm, that is, a permutation.
TableauARendererDOM and TableauAPairRendererDOM are the drawing classes.

TypeA/TableauRSAnimate.js

This file contains two classes, TableauAAnimate and TableauARendererAnimate.
The TableauAAnimate class's function is to execute one step of the Robinson-Schensted algorithm in such a way that the Tiles are not moved, but, instead, the new position of every Tile which would be moved is stored in the Tile. The TableauARendererAnimate class then renders the tableau and creates an animation function for every Tile which is to be moved.

TypeBCD/Tableau.js

This file stores the classes needed to perform the Domino Robinson-Schensted algorithm (and its inverse) and to draw the results.
The main class is the Tableau class, which handles domino tableaux.
The Domino class stores the information about the contents, location, and orientation of one domino in a domino tableau.
The DominoGrid class is a helper class for the Tableau class.
The TableauPair class holds a pair of Tableau.
The ParameterDominoRS class handles the input to the Domino Robinson-Schensted algorithm, that is, a signed permutation.
TableauRendererDOM and TableauPairRendererDOM are the drawing classes.
The TableauWithGrid class is an extension of the Tableau class. Domino tableaux are used for simple Lie groups of types B, C, and D. There is one grid for each type.

  • 2016-2017 Devra Garfinkle Johnson, 2016 Christian Johnson
