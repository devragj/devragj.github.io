Class: TableauSignsSUpq

TableauSignsSUpq(tableauopt, signsopt)

This class stores the output to the SU(p,q) algorithm. The output is usually represented as a pair of tableaux, one with numbers and one of the same shape, filled with signs. However, since, in the sign tableau, signs alternate along rows, all the information about the sign tableau is stored in the last square of each row. So, this class uses an array storing those last signs to represent the sign tableau. The method TableauSignsSUpq#getTableauPair converts this array into the corresponding sign tableau.
