Class: TableauCycles

TableauCycles(table)

This class extends TableauWithGrid. It adds methods needed to create animations of cycles.

Constructor

new TableauCycles(table)

Parameters:

table Object The first two inputs to the constructor are used to construct a TableauWithGrid.

 If true, the change takes place on the original tableau, otherwise, clones are created and shown.

getTableauToAnimateMoveOpenCycle(backSquare, noClone) → {Object}

This function optionally clones the tableau, and then computes the procedure of moving through a cycle specified by its back square on the tableau (cloned or original). However, instead of moving the Dominos, it stores the information of how the Domino would move (slide or rotate) in the Domino, for each Domino that would have been moved.

insertAtEnd(domino)

This function adds a Domino to the Tableau, in a situation where the domino can be added to the end of Tableau#dominoList. That is, either the domino is being added to a Tableau which contains numbers, and the number in the new domino is larger than any numbers already present in the Tableau, or, alternatively, the tableau's dominos do not contain numbers.
