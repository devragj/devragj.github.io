Class: TableauPairCycles

TableauPairCycles(type, left, right, animationDuration, inPlaceopt, unboxedOnlyopt)

This class extends TableauPairGrid. It adds methods needed to create animations of extended cycles.

Constructor

new TableauPairCycles(type, left, right, animationDuration, inPlaceopt, unboxedOnlyopt)

The first two inputs to the constructor are used to construct a TableauWithGrid.
Parameters:
Name Type Attributes Description
type string B, C, or D
left Tableau the left tableau of the pair
right Tableau the right tableau of the pair
animationDuration number how long, in milliseconds, the movement of each Domino should take.
inPlace boolean <optional>
 If true, the change takes place on the original tableau pair, otherwise, clones are created and shown.
unboxedOnly boolean <optional>
 If true, only special squares for unboxed cycles will be shown. Current uses also have inPlace == true.

getTableauPairToAnimateMoveOpenExtendedCycleLeft(backSquare) → {Object}

This function optionally clones the tableau pair, and then computes the procedure of moving through an extended cycle specified by the back square of one of its cyles in the left tableau, on the tableau (cloned or original). However, instead of moving the Dominos, it stores the information of how the Domino would move (slide or rotate) in the Domino, for each Domino that would have been moved.
