Class: ParameterSppq

ParameterSppq(inputopt)

This class handles the input to the Sp(p,q) algorithm. This parameter is an arrangement of the numbers 1 through n (where n = p + q) into pairs and single numbers, where each single number is associated with a sign, + or -. In addition, a pair of numbers will also have a sign (or orientation) associated with it. The parameter is stored in the array ParameterSppq#parameter. A parameter can also be represented as a string, and is for display purposes. So, this class contains functions to go between the two representations of the parameter.

Members

parameter :Array

This is an array which stores a representation of the parameter which is convenient for the algorithm. It contains numbers and sign strings ('+' and '-'). Functionally, the array is 1-based, that is, parameter[0] = 0, (basically just to discard that entry).
