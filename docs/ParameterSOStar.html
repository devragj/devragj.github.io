Class: ParameterSOStar

ParameterSOStar(inputopt)

This class handles the input to the SO*(2n) algorithm. This parameter is an arrangement of the numbers 1 through n into pairs and single numbers, where each single number is associated with a sign, + or -. In addition, a pair of numbers will also have a sign (or orientation) associated with it. The parameter is stored in the array ParameterSOStar#parameter. A parameter can also be represented as a string, and is for display purposes. So, this class contains functions to go between the two representations of the parameter.

input Array | string <optional>
 The input to this constructor can either be an array of the type required for ParameterSOStar#parameter, or it can be the string representation of the parameter. If input is not given, the constructor returns the representation of the empty parameter.
