This page is used to provide online access to the algorithm pages in my other repos, as well as to the repos themselves. Clinking on a link to a page will open the page, so that you can use it online. Clicking on a README link will open, in a new tab, the repo which contains the page. The README for the repo has instructions for the operation of the page. You can download the files from this repo (GitHub Pages Repository) or from the individual repos. Files with the same name in different repos are identical.
